# 🗳️ Chunav – Indian Election Assistant

An interactive web-based assistant designed to simplify and guide users through the Indian election process — from understanding eligibility to experiencing election day.


## 🚀 Features

* 🧭 **Election Journey**
  Step-by-step walkthrough of the entire election process — from eligibility to final results.

* 📅 **Timeline Overview**
  Dynamic election phases and key dates powered by Google Sheets.

* 🗳️ **Election Day Simulator**
  Simulates the real voting experience — from entering a polling booth to casting your vote.

* 🤖 **Interactive Assistant**
  Answers common election-related questions in a clear and user-friendly way.


  
## 🧠 Approach & Logic

The project focuses on **simplifying complex electoral processes** into an intuitive and structured experience.

* Information is broken into **guided flows (Journey + Simulator)**
* Timeline data is **dynamic and externally managed** via Google Sheets
* The assistant uses **rule-based intelligent responses** for:

  * reliability
  * fast performance
  * consistent user experience

The architecture is designed to be easily extendable with real-time AI APIs like Gemini.



## 🛠️ Tech Stack

* HTML, CSS, JavaScript
* Google Sheets (dynamic data source)
* Netlify (deployment)



## ⚙️ How It Works

1. Users navigate through structured sections via sidebar
2. Timeline data is fetched from Google Sheets
3. The assistant processes user queries and returns relevant responses
4. UI is designed for clarity, accessibility, and ease of use



## 📌 Assumptions

* Focused on Indian election system
* Designed primarily for first-time voters
* Prioritizes clarity over technical complexity
* Assistant responses are optimized for consistency
